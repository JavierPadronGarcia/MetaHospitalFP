Download data in excel

When you click on any of the buttons similar to this one, the next button will automatically download an excel document in .xlsx format and with an identifying name of the data included in the file.


In the specific case that you want to download the notes log, all the notes of the group will be downloaded and inserted into a single sheet called "General Information", in addition, this has a specific functionality, being able to filter data.


This functionality has full compatibility with two excel reading programs (Microsoft Excel and Google Calc), from those two programs you will automatically have the auto-filter created correctly.


In the case of Libre Office Calc it is not fully compatible, so in order to be able to filter all the content it is necessary to manually create the auto-filter in the following way:


Once the notes document has been downloaded, it will have a name similar to the following: in which you can see the group of grades and the date of generation of the excel document.

Next we create the automatic filter, for this we select the header row of the table as shown in the following image: 




With the cells selected there are several options for assigning the automatic filter:


- With the combination of keys (Ctrl + Shift + L).


- By accessing the top menu as shown in the following image:




- Accessing from the Data tab at the top of the window:



Once the action is done, some tabs will be added in each of the columns of the table, where you can click and make any filter by any of the columns of the table.




In this way, the automatic filter will be done in Libre Office Calc and you will be able to make all the filters you need.
