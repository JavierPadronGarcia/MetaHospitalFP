Teachers

As a teacher, the first interface you will see are all the groups to which you are assigned, as shown in the image below.


When you access one of the groups you will see a panel in which you can navigate to two sections, to see the students and the work units, where you will see the grades.




STUDENTS:



In the students view you can see the students assigned to the selected group, in this case 1ÂºCAE.


Among the actions that can be performed are:


- Reset the password in case the user forgets it (padlock icon) in which the default password "metahospital" will be assigned.



- Assign students to the group in case they are not assigned, if the user is created they can be assigned to the group by clicking on the icon in the lower right corner. 


-Un-assign the student from the group, so he/she will no longer be in that group and will not be able to access it in his/her account, this is done with the bin icon in the "Actions" column . 



In addition, you can also see the code with which the students access to do the exercises




UNITS:


In the units view you can see two views of grades, for more details about this view go to Viewing grades.
