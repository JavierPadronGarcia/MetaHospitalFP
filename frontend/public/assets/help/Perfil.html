Perfil

Códigos de autenticación para la plataforma en VR:


Desde esta página es donde podrá ver su código de autenticación con el que podrá acceder a las unidades de trabajo de MetahospitalFp donde podrá realizar casos y podrá ser identificado para registrar las calificaciones en su usuario.



El código se genera automáticamente cuando se le asigna en un grupo pero en caso de que el código expire podrá volver a generar un código nuevo con el que acceder


Actualización / Restablecimiento de contraseña:


Si quiere actualizar su contraseña por motivos de seguridad también puede realizarlo desde su perfil donde podrá escribir la nueva contraseña y confirmarla, una vez actualizada, el siguiente inicio de sesión que realizará debe ser con la contraseña nueva.


En caso de que se olvide la contraseña se deberá avisar al profesor del grupo correspondiente en el caso de ser estudiante, en otro caso se avisará a un administrador para que reinicie la contraseña.


Personalización del perfil:


En cuestiones de personalización se puede actualizar la imagen de perfil y elegir una imagen del dispositivo y se actualizará, para ello se hace clic en la imagen del perfil y aparecerá el explorador de imágenes:



Por defecto tendrá esta imagen, pero si se personaliza aparecerá la imagen correspondiente


Cambiar idioma:


En caso de necesitar cambiar de idioma también se puede cambiar desde el perfil, en la esquina inferior derecha estará una imagen con el idioma actual, si se clica, se desplegarán todos los idiomas soportados y se pueden seleccionar para cambiar el idioma, además de la página del perfil se puede actualizar el idioma desde el inicio de sesión, el idioma seleccionado es el primero desde la parte inferior, en este caso el español es el idioma seleccionado.
